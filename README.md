# awx-vmware-python

* Custom python script for VMware dynamic inventory

## Last Review

* June/2020

## Required

* Libs:

```
- pyVmomi
```

* ENVIROMENT Variables:

```
- VMWARE_USERNAME
- VMWARE_PASSWORD
- VMWARE_SERVER
- VMWARE_VALIDATE_CERTS (true or false)
```
